## ğŸ“¦ Installation & Usage

### Quick Start

1. **Clone the repository:**
```bash
git clone https://github.com/dddevid/Business-websites-templates.git
cd Business-websites-templates
```

2. **Choose a template:**
```bash
cd [template-name]
```

3. **Open in browser:**
```bash
# Simply open index.html in your browser
# Or use a local server (recommended)
python -m http.server 8000
```
